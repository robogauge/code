Anonymous integration code for the paper "Toward Reliable Sim-to-Real Predictability for MoE-based Robust Quadrupedal Locomotion". Video and project page can be found at: [https://robogauge.github.io/](https://robogauge.github.io/)

1. [go2_rl_gym](./go2_rl_gym/README.md): Reinforcement learning environment for Go2 robot with MoE CTS algorithm.
2. [RoboGaguge](./RoboGauge/README.md): A suite of motion-control evaluation for reinforcement-learning locomotion policies.
3. [unitree_cpp_deploy](./unitree_cpp_deploy/README.md): Deployment of Unitree robots using C++.
